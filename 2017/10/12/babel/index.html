babel教程

  1. 介绍
    babel是一个广泛使用的转码器，可以将ES6代码转为ES5代码。

  2. 配置文件.babelrc

    新建.babelrc，打开文件写入：

    {
    “presets”: [“es2015”],
    “plugins”: []
    }

    其中es2015是指“ES2015转码规则”，

    需要安装 npm install –save-dev babel-preset-es2015

  3. 命令行转码

    Babel提供babel-cli工具，用于命令行转码。

    npm install –global babel-cli

  4. 用法

    在项目中使用

    1. npm install –save-dev babel-cli

    2. npm init -y 生成package.json文件，改写文件内容为：
      {
      // …
      “devDependencies”: {
      “babel-cli”: “^6.0.0”
      },
      “scripts”: {
      “build”: “babel src -d lib”
      },
      }

      其中：src文件里放你要解码的文件，它会自动生成lib文件，将解码之后的文件放到lib文件夹里

    3. 运行
      npm run build
